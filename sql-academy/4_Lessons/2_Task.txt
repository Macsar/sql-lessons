// ------------ ЗАДАНИЕ ------------ //
Найдите самых старших членов семьи (используйте поле birthday) среди всех существующих семей
на основании их статуса (поле status). Выведите статус и дату рождения.
Для вывода даты рождения используйте псевдоним birthday.


// ------------ РЕШЕНИЕ ------------ //
SELECT
    status, MIN(birthday) AS birthday
FROM
    FamilyMembers
GROUP BY
    status
