// ------------ ЗАДАНИЕ ------------ //
Получите среднее время полётов, совершённых на каждой из моделей самолёта. Выведите поле plane
и среднее время полёта в секундах.
Для вывода времени используйте псевдоним time.
Используйте функцию TIMESTAMPDIFF(second, time_out, time_in), чтобы получить разницу во времени
в секундах между двумя датами.


// ------------ РЕШЕНИЕ ------------ //
SELECT
    plane, AVG(TIMESTAMPDIFF(second, time_out, time_in)) AS time
FROM
    Trip
GROUP BY
    plane
