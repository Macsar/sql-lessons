Для каждого отдельного платежа выведите идентификатор товара и сумму, потраченную на него,
в отсортированном по убыванию этой суммы виде. Список платежей находится в таблице Payments.
Для вывода суммы используйте псевдоним sum.

// ---------- || ---------- //

SELECT good, amount * unit_price AS sum FROM Payments
ORDER BY sum DESC;
